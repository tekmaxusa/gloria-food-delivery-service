## Option 2: Sa 2-Step Verification Page

1. **Scroll down** sa 2-Step Verification page
   - Ang "App passwords" ay nasa **ilalim** ng listahan ng second steps
   - Baka kailangan mong i-scroll pababa para makita

2. **Hanapin ang "App passwords"** section
   - Dapat may nakalagay na "App passwords" o "App-specific passwords"
   - I-click ito

---

## Option 3: Sa Security Settings

1. Pumunta sa: https://myaccount.google.com/security
2. Hanapin ang **"2-Step Verification"** section
3. I-click ang **"2-Step Verification"**
4. Sa 2-Step Verification page, **scroll down** at hanapin ang **"App passwords"**

---

## Kung Hindi Makita ang "App passwords"

### Kailangan naka-enable ang 2-Step Verification

Kung wala pa naka-enable na second step method:
1. I-enable mo muna ang kahit isa:
   - **Google prompt** (recommended - gamit ang phone)
   - **Phone number** (SMS)
   - **Authenticator app**

2. Pagkatapos ma-enable, lalabas na ang "App passwords" option

## Quick Steps:

1. **Pumunta sa:** https://myaccount.google.com/apppasswords
   - O i-type: `myaccount.google.com/apppasswords`

2. **Kung hindi makita:**
   - I-enable muna ang 2-Step Verification
   - I-enable ang Google prompt o Phone number
   - Pagkatapos, lalabas na ang App passwords option

3. **Generate App Password:**
   - Pumili ng app: "Mail" o "Other (Custom name)"
   - Pumili ng device: "Other (Custom name)"
   - Lagyan ng name: "GloriaFood Webhook"
   - I-click "Generate"
   - **Kopyahin ang 16-character password**

---

## Alternative: Gamitin ang Search

1. Pumunta sa: https://myaccount.google.com/
2. I-type sa search box: **"app passwords"**
3. I-click ang result na "App passwords"

---

## Troubleshooting

**"App passwords" hindi makita kahit naka-enable ang 2-Step Verification?**
- Try mo i-refresh ang page
- O pumunta direkta sa: https://myaccount.google.com/apppasswords
- O i-logout at i-login ulit

**"This feature isn't available" error?**
- Baka hindi pa fully activated ang 2-Step Verification
- Try mo i-enable ulit ang Google prompt o phone number
- Wait ng ilang minutes bago i-try ulit
